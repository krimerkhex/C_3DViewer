3DViewer

Общее введение

Данный проект направлен на визуализацию 3D объектов.

Функционал приложения

Выбор файла для отображения.
Смещение отображаемого объекта по осям x,y,z.
Аффиные преобразования объекта по осям x,y,z.
Увеличение и уменьшение маштаба объекта.
Изменение цвета заднего фона.
Изменение цвета вершин.
Изменение цвета полигонов.
Изменение типа проекции.
Изменение толщины вершин и полигонов.
Изменение типов полигонов и вершин.
Сознание скриншота в формате .bmp и .jpeg.
Отображение имени файла.
Отображение количества вершин и полигонов у объекта.

Make targets

Таrgets:
 ⥗  all - запуск калькулятора и открытие исполняемого объекта
 ⥗  install - таргет для начала работы с калькулятором
 ⥗  unistall - таргет для удаления программы
 ⥗  clean - очистка "мусорных" файлов
 ⥗  dvi - вывод паспорта проекта
 ⥗  dist - таргет для сборки QT
 ⥗  test - таргет проверки backend-а с помощью unit_test.c
 ⥗  gcov_report - отображение покрытия backend-а unit тестами
